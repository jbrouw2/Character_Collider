# Character Collider

### Generate a cast of characters with colliding personalities!

- Run a simple Python script from the command line
- Choose your total number of characters
- Choose the number of characters with High scales on the HEXACO Personality Inventory
    - **H** - Honesty-Humility
    - **E** - Emotionality
    - **X** - eXtroversion
    - **A** - Agreeableness
    - **C** - Conscientiousness
    - **O** - Openness to Experience
